Miracle - a calm a cozy mental health web-app

Features
1. AI chatbot
2. Mood tracking
3. Digital diary
4. Virtual pet

Requirements 
1. Node.js (v16+ recommended)
2. npm (comes with node)

Respositoriesadd
1. Frontend
https://github.com/yuitxin-boop/super-stressed-app
2. Backend
https://github.com/yuitxin-boop/miracle-backend 

Install dependencies
npm install

Run the app in development mode
npm start - Open http://localhost:3000 to view it in your browser.

Running test
npm test - launches the test runner in the interactive watch mode.

Building for production
npm run build - builds the app for production to the build folder.
It correctly bundles React in production mode and optimizes the build for the best performance.

Start backend server
1. cd backend
2. node server.js
Opens http://localhost:5000 to view in your brower

Deployment
Frontend (React)
1. Deployed using vercel
2. Live link: https://super-stressed-app-zeta.vercel.app/
3. Automatically redeploys when changes are pushed to the respository

Backend (Node.js)
1. Deployed using Render
2. API link: https://miracle-backend-eq4m.onrender.com
3. Make sure the frontend uses this deployed API URL when making requests
